## <img src="https://readme-typing-svg.herokuapp.com?font=Lexend+Giga&size=25&pause=1000&color=9370DB&vCenter=true&width=435&height=25&lines=INSTALLATION" width="450"/>

> ### Direct Installation
> You can now easily start the script directly without cloning the repository. Just copy and paste the command bellow in your Terminal (Admin)  and run it..

```
curl -s https://raw.githubusercontent.com/stquremwork/primus-tl/main/script.sh | bash
```
---
### Manual Installation

1. Clone this repository
```
git clone https://github.com/stquremwork/cirrus-tl.git
```

2. Navigate to the repository folder
```
cd cirrus-tl
```

3. Make the script executable
```
chmod +x script.sh
```

4. Run the script
```
./script.sh
```
